# 如何配置 SSH 公钥访问 git 仓库

  1. 本地用ssh-keygen生成私钥

    打开命令行终端输入ssh-keygen -t rsa -C <your_email@example.com>( 你的邮箱)，连续点击 Enter 键即可。

  2. 公钥配置到git服务器

# git push时不需要输入密码

如果你的git仓库拉到本地时用的是 https 的协议，那么在不做任何操作的情况，git push 时是需要输入 git 的用户名和密码的。

如何达到git push 时不需要输入 git 的用户名和密码呢？

你可以：

  • 走 SSH 协议
    • 即你在clone文件时即使用ssh协议
  • 走 https 协议
    • 配置 credential helper
